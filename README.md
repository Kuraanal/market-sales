## How to Use

### On Desktop
1. Open `index.html` in your web browser
2. Start adding items and tracking sales!

### On Mobile
1. Visit the GitHub Pages URL: `https://YOUR_USERNAME.github.io/market-sales/`
2. Add to home screen for app-like experience
3. Works offline after first load!

## Getting Started

1. Click the **+** button to add your first product
2. Tap products to add them to your sales
3. View **Summary** to see your daily breakdown
4. Click **End Day & Clear** to save history and start fresh

## Tech Stack

- Pure HTML5, CSS3, and JavaScript
- No frameworks or dependencies
- Uses browser localStorage for data persistence
